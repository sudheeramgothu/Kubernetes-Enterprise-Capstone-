# Weeks 3–4 – Cluster Operations & Troubleshooting (CKA Heavy)

## Objective
This phase prepares you for the most heavily tested CKA topics: resource management,
scheduling constraints, autoscaling, and troubleshooting failing workloads.

## Topics
- Resource requests and limits
- Scheduling, taints & tolerations
- Horizontal Pod Autoscaler
- Debugging failing workloads

## Deliverables
- Scalable workloads
- Resource isolation
- Troubleshooting report

## CKA Exam Mapping
- Resource management
- Scheduling
- Autoscaling
- Troubleshooting
